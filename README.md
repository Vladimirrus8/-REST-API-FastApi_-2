# Advertisement Service API

Сервис объявлений купли/продажи с авторизацией и системой ролей.

## Технологии

- **FastAPI** - веб-фреймворк
- **SQLAlchemy** - ORM
- **PostgreSQL** - база данных
- **JWT** - аутентификация
- **Docker** - контейнеризация
